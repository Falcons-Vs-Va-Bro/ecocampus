# EcoCampus Frontend

EcoCampus Web/H5 frontend skeleton built with React, TypeScript, Vite, pnpm, React Router, TanStack Query, Zustand, Axios, Tailwind CSS, Ant Design, lucide-react, and Motion for React.

## Scripts

```bash
pnpm install
pnpm dev
pnpm lint
pnpm build
```

## Notes

- Routes are defined in `src/app/routeCatalog.ts`.
- API wrappers use `/api/v1` by default and can be overridden with `VITE_API_BASE_URL`.
- Current pages are contract-ready placeholders. Business UI should be implemented under `src/features/`.
